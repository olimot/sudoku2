How to use user interface

  1. Click ◀ or ▶ to change the level.
  2. Click start button to generate and play a game in the level shown at the label of button.
  3. Click a number button below or simply press a digit key in keyboard to select it.
  4. Toggle note mode by clicking a selected number again or clicking 'Note (space)' button.
  5. Click a cell to put or note the number selected.
